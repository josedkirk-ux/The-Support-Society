# The Support Society

"Celebrating everyday heroes… just like YOU!"

A website dedicated to honoring and supporting parents and guardians of children with disabilities.

## About This Project

The Support Society is a community-focused website built to celebrate and support the everyday heroes who care for children with disabilities. This is a beginner-friendly web project built with HTML and CSS.

## Project Structure

```
the-support-society/
├── index.html       # Main page
├── styles.css       # Styling
└── README.md        # This file
```

## Features

- Clean, professional design
- Responsive layout
- Easy navigation
- Multiple sections (About, Resources, Stories, Events)
- Footer with social media links

## Getting Started

1. Open `index.html` in your web browser to view the website locally
2. Edit `index.html` to add content
3. Edit `styles.css` to customize the appearance
4. Once ready, deploy to GitHub Pages (see instructions below)

## Deploying to GitHub Pages

1. Go to your repository settings
2. Scroll down to "GitHub Pages"
3. Select "main" branch as the source
4. Your website will be live at: `https://josedkirk-ux.github.io/the-support-society/`
